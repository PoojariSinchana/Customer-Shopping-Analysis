---------------Customer Shopping Behavior Analysis-------------------
-------Project Overview
This project analyzes customer shopping behavior using Python and Power BI.

The goal is to identify purchasing trends, customer segments, revenue patterns, and business insights that help improve decision-making.

-----Tools Used
- Python
- Pandas
- NumPy
- Power BI
- Excel
- Git
- GitHub

------Dataset
The dataset contains customer shopping transactions with information such as:

• Customer ID
• Age
• Gender
• Product Purchased
• Category
• Purchase Amount
• Season
• Shipping Type
• Payment Method
• Review Rating
